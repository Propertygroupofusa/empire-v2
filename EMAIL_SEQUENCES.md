## Implementation Instructions

### Where to Send These:

1. **Setup in your email platform** (ConvertKit, ActiveCampaign, MailerLite):
   - Create automation/sequence for each path
   - Set conditional delays between emails
   - Tag subscribers by path (signals, api, dfy, whitelabel)

2. **Triggers:**
   - Path 1: Send when email subscribes + adds trading signals to cart
   - Path 2: Send when API key is created
   - Path 3: Send when strategy call is booked
   - Path 4: Send when application is submitted

3. **Personalization:**
   - Use [Name] → First name
   - Use [Company] → Company name
   - Use [LINK] → Unique tracking link for each path

4. **A/B Testing:**
   - Test subject lines
   - Test email length (shorter vs detailed)
   - Test CTA placement and wording
   - Track: Open rate (target 40%+), Click rate (target 8%+), Conversion (target 2-5%)

### Expected Results:

**Path 1 (Trading Signals):**
- Email 1 opens: 60%
- Email 7 conversion: 8-15% (to paid subscription)
- LTV: $297 × 12 = $3,564/customer

**Path 2 (API/SaaS):**
- Email 1 opens: 50%
- Email 5 conversion: 12-20% (upgrade from free to paid)
- LTV: $299 × 8 (avg tenure) = $2,392/customer

**Path 3 (Done-For-You):**
- Email 1 opens: 55%

### Pro Tips:

1. **Personalize subject lines** - Use [Name] for 10-20% open boost
2. **Send from a real person** - "From: Del" converts better than "From: Team"
3. **Mobile optimize** - 60%+ of opens are on mobile
4. **Track everything** - Which emails convert best? Double down on those
5. **Update with results** - When someone from Path 3 gets results, mention it in Path 1 emails
6. **Segment by behavior** - Non-openers get different sequence than high-engagers
7. **Unsubscribe is okay** - People who aren't interested shouldn't stay on list
